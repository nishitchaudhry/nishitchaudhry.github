Projects


Voice based home automation


Voice based switching and intensity control of Incadescent and LED lights 
using Amazon Alexa AI over bluetooth. Also, a smart plug was designed to 
implement switching control of electrical appliances to replace original 
sockets with plug and play style.

Microntroller used : Arduino uno
Language used : C
				
Microprocessor used : Raspberry pi 3
Language used : Python
				

Smart blind stick


Blind stick was designed using a microcontroller environment with various 
sensors like ultrasonic, water, light. A special feature of GSM+GSP was 
integrated to let the care taker know the location of the blind person 
which is communicated over GSM.

Language used : C
Microntroller used : STM32
